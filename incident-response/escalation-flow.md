# Escalation Workflow

## Severity Levels

| Severity | Action |
|---|---|
| Low | Monitoring |
| Medium | Analyst review |
| High | Security escalation |
| Critical | Incident response activation |

## Escalation Process

Detection
    ↓
SIEM Correlation
    ↓
Security Analysis
    ↓
Incident Escalation
    ↓
Containment & Recovery
