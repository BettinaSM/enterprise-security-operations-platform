# Incident Response Playbook

## Detection

Security alerts are generated from runtime detections and SIEM correlation.

## Triage

- Validate alert severity
- Identify affected assets
- Verify suspicious activity

## Containment

- Restrict affected workload
- Isolate suspicious container
- Block malicious activity

## Investigation

- Review logs
- Analyze MITRE techniques
- Validate indicators of compromise

## Recovery

- Restore secure environment
- Validate remediation
- Monitor post-incident activity
